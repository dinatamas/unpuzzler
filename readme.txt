Unpuzzle - Automatic Puzzle Solver
==================================

The `unpuzzler.py` solves jigsaw puzzles.

Tasks
-----

  - Automate the detection of the number of puzzle pieces.

  - Option to remove green background from images.

  - Script to combine multiple files into one.

ImageMagick:

`montage *.jpg -geometry 120x120>+15+15 combined.jpg`
